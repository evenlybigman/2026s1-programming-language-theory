### 프로그래밍 언어의 역사

#### Fortran

- 최초의 고급 언어

- 1954년 개발

- 특징

  - 효율성 (= 빠른 실행)

  - 정적 (static) 메모리 관리

대문자로 작성

#### Cobol (Common business oriented language)

- 업무용 언어(기업에서 사용)
- 1959 ~ 1960
- 비즈니스 관련 시스템 개발에 이용됨 
- 특징
  - 낮은 가독성
  - Record 도입
  - 선언부 + 실행부

#### Algol

- 범용(general-purpose) 프로그래밍 언어
- 1950년대 말 ~ 60년대 초
- 특징
  - 블록 구조
  - 매개변수/파라미터/인자 전달

#### Pascal

- 특징
  - 프로그래밍 교육용
  - 블록 구조
  - 단순

#### C

- 1972
- UNIX  운영체제 (operation system) 개발에 이용됨 
- 특징
  - 단순, 간결
  - 효율적 (빠른 실행)
  - 포인터 데이터 유형
- 단점
  - 낮은 가독성
  - 약한 타입 검사
  - 데이터 캡슐화 기능 X

#### C++

- Simula 67 + C + class
- 특징
  - 객체지향 언어
  - 데이터 캡슐화 지원

#### C#

- console application
- winform application
- 2000
- 객체지향 언어
- 컴포넌트 기반 프로그래밍
- 특징
  - 강한 타입 검사
  - 배열 경계 검사
  - 초기화하지 않은 변수 탐지
  - 자동 쓰레기 수집
  - 국제화 : unicode

#### Java

- Java platform = Java VM + Java API
- 하이브리드 방식 언어
- 특징
  - 독립적임
  - c++보다 단순
  - 분산 환경 지원
  - 객체지향 언어

#### Perl

- CGI 프로그램 개발에 이용
- 텍스트 파일 처리에 강점
- 인터프리터 언어
- 두 가지의 변수
  - 스칼라 형: $
  - 배열형 : @
- 특징
  - 변수 선언 X
  - 배열 크기 미리 선언 X
  - 디폴트 초기값 제공

#### Lisp

- List Processor
- 함수형 언어
- 인공지능 프로그래밍 언어
- 특징
  - 자동 기억 장소 관리
  - 동적 바인딩

#### 다른 언어들

- 그 외 SNOBOL, BASIC, 등등
