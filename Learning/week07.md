# 1day report

## Gemini CLI

- 웹 기반 도구의 한계 해결
- 반복적인 복사/붙여넣기 작업 제거
- 작업 중 창 전환 없이 개발 가능
- 개발 과정 전체를 하나로 통합
- 자동화된 개발 환경 제공

## Gemono CLI 사용 준비

- API 키 생성
- CLI 도구 설치
- 환경 변수 설정

## 주요 명령어

- 기본 질문
  - gemini "질문"
- 파일 설명
  - gemini explain 파일명
- 파일 기반 질문
  - gemini "질문" -f 파일명

## 효과적인 프롬프트 작성

- 페르소나 + 작업 + 상황 + 출력형식
- 구체적으로 작성할수록 더 정확한 결과 생성

## 확장 기능

- /memory : 이전 대화 기억 및 활용
- -f : 파일을 읽어서 분석
- /chat : 대화 저장 및 불러오기
- /copy : 결과 코드 복사

## AI 에이전트 동작 방식

- Plan : 요청 분석 및 해결 방법 결정
- Act : 코드 실행 및 처리
- Observe : 결과 확인 후 수정 반복

## AI 활용 방법

- 구체적으로 질문하기
- 결과를 무조건 믿지 말고 검증하기
- 추가 질문으로 결과 개선하기
